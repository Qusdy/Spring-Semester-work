# Описание проекта

Новостная платформа — это веб-приложение для создания, публикации и обсуждения новостных постов в тематических группах. Проект представляет собой полнофункциональную социальную платформу для авторов и читателей, где пользователи могут создавать группы, публиковать посты, оставлять комментарии и следить за новостями через внешние API.

# Технологический стек

## Backend
- Java 17 + Spring Boot 3.4.4
- Spring MVC, Data JPA, Security, AOP
- PostgreSQL
- Redis
- Hibernate (JPA)
- Liquibase
- Gradle

## Frontend
- Freemarker (шаблонизатор)
- JavaScript + jQuery
- Bootstrap 5 (стилизация)

## Интеграции
- Cloudinary (хранение изображений)
- NewsAPI (внешние новости)
- Docker Compose (контейнеризация)
- OpenAPI Generator (генерация API)

# Основные возможности
## Пользователи
- Регистрация и аутентификация (Spring Security)
- Роли: USER, MODERATOR, ADMIN
- "Запомни меня" (Remember-Me)
- Управление пользователями (администратор)

## Группы
- Создание тематических групп
- Загрузка логотипа (Cloudinary / локальное хранилище)
- Назначение владельца группы
- Редактирование и удаление

## Посты
- Создание, редактирование, публикация
- Черновики (DRAFT) и публикация (PUBLISHED)
- Загрузка изображений 
- Теги для категоризации 
- Подсчёт просмотров 
- Поиск по заголовку и содержимому

## Комментарии
- Добавление комментариев к постам
- Редактирование и удаление (автор или модератор)
- Отметка об редактировании 
- Уведомления авторам

## Уведомления
- Уведомления о новых комментариях
- Отметка о прочтении
- Счётчик непрочитанных

## Внешние новости
- Интеграция с NewsAPI
- Поиск внешних новостей
- Кэширование через Redis

# Запуск

## Настройка окружения
Создайте файл .env:
```env
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLOUDINARY_ENABLED=
NEWS_API_KEY=
```

некоторые элементы могут быть не указаны (например cloudinary, тогда файлы будут хранится локально)

## Соберите проект

```bash
./gradlew build
```

## Запустите

```bash
docker-compose up -d
```
